# Module 2: High-Performance Psychology (People)

**Theme:** Engineers are deterministic. Humans are not. Debug the root cause.

---

This module focuses on the most unpredictable part of the stack: Human Beings. You'll learn how to conduct meaningful one-on-ones, deliver effective feedback, address underperformance, and build diverse high-performing teams.

## What You'll Learn

### 2.1 One-on-Ones: The Anti-Status Update
Learn how to transform 1:1s from status updates into the critical maintenance sessions for your most expensive asset: your engineers.

### 2.2 Radical Candor for Introverts
Master the art of giving direct, actionable feedback using the SBI framework without being "ruinously empathetic."

### 2.3 Debugging Underperformance (The PIP)
Understand when and how to address underperformance using the Will vs. Skill framework before high performers start quitting.

### 2.4 Hiring: "Culture Add" vs. "Culture Fit"
Stop hiring people who look like you and start raising the bar by seeking diversity in thought and capability.
